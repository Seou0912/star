# 주요 기능

### 로그인 , 회원가입 
로그인이나 회원가입을 하면 오늘의 한마디로 화면이 전환됩니다.<br>

>#### 오늘의 한마디: <br>
>`별자리운세`, `띠별운세`, `MBTI 운세` <br>

### 여행 지점들의 일자별 최적 동선 제공

사용자의 생년월일로 별자리와 띠에 대한 운세를, <br>
OpenAi 기반으로 제공합니다. <br>
